HTML Inline Text Formatting

This page demonstrates various inline text tags used to format text and add semantic meaning or visual styles.

1. Strong Importance

This text is wrapped in the <strong> tag.

Label: The <strong> element is a semantic tag that indicates the text has strong importance, seriousness, or urgency. Browsers render it bold by default, and screen readers emphasize it audibly.

2. Visual Bold

This text is wrapped in the <b> tag.

Label: The <b> element is a presentational tag used to make text bold visually for drawing attention, without implying any semantic importance or extra weight.

3. Semantic Emphasis

This text is wrapped in the <em> tag.

Label: The <em> element is a semantic tag indicating emphasis (stress) on a word or phrase, which alters the spoken meaning of the sentence. It renders in italics by default.

4. Visual Italic

This text is wrapped in the <i> tag.

Label: The <i> element is a presentational tag used to italicize text visually for styling purposes, such as defining foreign terms, book titles, thoughts, or technical terms, without adding emphasis.

5. Highlighted Text

The deadline to submit the report is this Friday at noon.

Label: The <mark> element is a semantic tag used to highlight or mark text for reference or relevance, representing a yellow highlighter effect by default.

6. Small Print

Terms and conditions apply. All rights reserved.

Label: The <small> element is a semantic tag used to represent side-comments, fine print, disclaimers, or copyright text, rendering one size smaller than the body text.

7. Subscript

The molecular formula of glucose is C6H12O6.

Label: The <sub> element defines subscript text, which is positioned half a character below the baseline and rendered in a smaller font, commonly used in chemistry and math formulas.

8. Superscript

Einstein's famous equation is E = mc2.

Label: The <sup> element defines superscript text, which is positioned half a character above the baseline and rendered in a smaller font, commonly used for exponents, footnotes, and ordinal indicators.
