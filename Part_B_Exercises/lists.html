HTML List Types

This page demonstrates the three primary list structures available in HTML, along with nested list structures.

1. Ordered List (<ol>)

Used when the order of items is sequential and matters:

  1. Turn on the computer and open a text editor.
  2. Write the HTML structure with correct semantic markup.
  3. Save the document with an .html file extension.
  4. Open the file in a web browser to verify the output.

2. Unordered List (<ul>)

Used when the items are related but the sequence does not matter:

  • HTML5 (for document structure)
  • CSS3 (for styling and layout)
  • JavaScript (for interactive behavior)
  • Git & GitHub (for version control and collaboration)

3. Description List (<dl>)

Used to list terms and their corresponding descriptions, like a dictionary or glossary:

HTML5
The fifth and current major version of the HyperText Markup Language standard, supporting semantic tags, native multimedia, and APIs.
CSS3
The modern version of Cascading Style Sheets, allowing for flexible layout systems (like Flexbox and Grid), transitions, and media queries.
DOM
Document Object Model. A programming interface for web documents that represents the page structure as a tree nodes, enabling dynamic scripting.
SEO
Search Engine Optimization. The practice of structuring and labeling web page content to increase its visibility in search engine result pages.

4. Nested List

A list nested inside another list item to represent sub-levels (e.g., a syllabus or directory structure):

    • HTML5 Basics
    • CSS3 Styling
    • JavaScript Programming
  • Backend Development

  • Mobile App Development

  • Developer Tooling
