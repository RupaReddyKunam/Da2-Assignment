# HTML5 Basics - Day 2 Assignment

This repository contains the complete solutions for the Day 2 HTML5 Basics assignment as outlined in the course material (Section 25, Page 92).

## Project Structure

The project is organized into the following directories:

*   **`Part_A_Theory/`**: Contains answers to the 10 theory questions about HTML, elements, attributes, accessibility, and SEO.
*   **`Part_B_Exercises/`**: Contains 5 practical exercises reinforcing HTML tags, structures, list layouts, and data tables.
*   **`Part_C_Mini_Project/`**: A complete Personal Portfolio page representing the final practical assignment. It includes semantic HTML5 layouts, anchor navigation, safe external links, and multimedia players.
*   **`Part_D_Research/`**: Contains findings from research on advanced HTML5 form elements, W3C/WHATWG standards, WCAG guidelines, and block vs. inline elements.
